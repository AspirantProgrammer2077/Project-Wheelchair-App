♿ Project Wheelchair App

A smart assistive mobility system that enables voice-controlled and Bluetooth-based navigation for electric wheelchairs using ESP32/Arduino hardware integration.

🚀 Core Features
- 🎤 Voice command navigation (Forward, Backward, Left, Right, Stop)
- 🎮 Manual control interface
- 📡 Bluetooth communication (ESP32 / HC-05)
- 🚧 Obstacle-aware movement system (ultrasonic sensor integration)
- ⚙️ Automatic and manual control modes

🛠️ Tech Stack
- Flutter (Mobile App)
- ESP32 / Arduino Uno
- HC-05 Bluetooth Module
- Ultrasonic Sensors
- BTS7960 Motor Driver

🔐 Safety Features
- Emergency stop system
- Manual override control
- Sensor-based obstacle detection

🎯 Purpose
To improve mobility and independence for individuals with physical disabilities through a low-cost IoT-based wheelchair control system.

📌 Note
This project is part of a research-based prototype and is intended for educational and assistive technology development purposes. This is only the front-end application.
